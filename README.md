# 📝 React Todo List

Приложение для управления задачами (Todo List), построенное на **React 19** с использованием **Vite**. Проект демонстрирует профессиональный подход к разработке: грамотную архитектуру по методологии Feature-Sliced Design, оптимизацию производительности, модульную стилизацию и чистый поддерживаемый код.

## 🎯 Обзор проекта

React Todo List — это полноценное приложение для управления задачами, созданное в рамках pet-проекта с целью демонстрации современных подходов к разработке на React. Приложение позволяет пользователям создавать, редактировать, удалять и отслеживать выполнение задач, а также выполнять поиск задачи по названию.

Проект особенно интересен своей архитектурой и вниманием к деталям: здесь реализована методология Feature-Sliced Design (FSD), проведена работа по оптимизации рендеринга, использована модульная стилизация и продумана структура кода, готовая к масштабированию.

---

## ✨ Ключевые особенности

### Основной функционал
- **➕ Создание задач** — добавление новых задач с текстовым описанием
- **✏️ Редактирование** — изменение текста существующих задач
- **🗑️ Удаление** — удаление ненужных задач из списка
- **✅ Отметка о выполнении** — переключение статуса задачи (выполнено/не выполнено)
- **🔍 Поиск** — поиск задач
- **📊 Счетчик задач** — отображение количества невыполненных задач
- **📄 Детальный просмотр** — отдельная страница с полной информацией о задаче
- **🔗 Маршрутизация** — навигация между списком задач и детальным просмотром

### Технические преимущества
- **🚀 Высокая производительность** — устранены лишние ререндеры, оптимизированы обновления списка
- **🏗️ Масштабируемая архитектура** — разделение кода по слоям согласно FSD
- **🎨 Модульная стилизация** — CSS Modules для изоляции стилей и предотвращения конфликтов
- **📁 Чистая структура** — логическое разделение на фичи, сущности и shared-компоненты
- **🛠️ Готовность к интеграции** — имитация API через JSON5, легкая замена на реальный бэкенд

## 🏛️ Архитектура проекта

Проект построен на методологии **Feature-Sliced Design (FSD)**, которая разделяет код на несколько независимых слоев. Это обеспечивает высокую связность внутри модулей и слабую связанность между ними, упрощая тестирование, поддержку и масштабирование.

## 🚀 Установка и запуск

### Системные требования
- **Node.js**: версия 18.0.0 или выше
- **npm**: версия 9.0.0 или выше (или yarn/pnpm)
- **Git**: для клонирования репозитория

### Пошаговая инструкция

```bash
# 1. Клонирование репозитория
git clone https://github.com/VaganovaIP/react-todo-list.git

# 2. Переход в директорию проекта
cd react-todo-list

# 3. Установка зависимостей
npm install
# или
yarn install

# 4. Запуск в режиме разработки
npm run dev
# или
yarn dev
